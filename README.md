# AI-Assisted Directory Log Analyzer
This project simulates an IAM automation workflow by analyzing directory service logs, detecting potential anomalies, and generating AI-powered summaries. It demonstrates skills relevant to identity management, automation, and applied AI.
## Features
- Parses directory-style log entries
- Detects unusual login patterns (failed attempts, unknown devices, rapid retries)
- Generates structured JSON reports
- Uses an AI summarization function to produce human-readable insights
- Beginner-friendly Python code

## Technologies Used
- Python
- JSON
- Basic automation logic
- AI-assisted analysis (placeholder function)

## How It Works
1. The script reads sample directory logs.
2. It identifies anomalies such as:
   - repeated failed logins
   - access from unknown IPs
   - rapid login attempts
3. It outputs a JSON report.
4. It generates an AI-style summary of the findings.
